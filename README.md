Virtual Banking System


It is a simple online banking system that lets users (customers and admins) manage banking tasks like signing up, logging in, checking balances, and viewing transaction history. It’s built using Java, Spring Boot, HTML, CSS, JavaScript, and MySQL.

Key Features:
- Signup & Login: Customers and admins can register and log in securely.
- Customer Dashboard: Customers can see their account details and current balance.
- Admin Dashboard: Admins have special access to manage users and view histories.
- Passbook Page: Shows all transactions with date and time, just like a real bank passbook.
- Deposit, Withdraw, Transfer: Customers can deposit money, withdraw funds, and transfer money to other users.
- User Profile: Customers can update their name, email, and password.
- Add Users: Admins can create new customer accounts.
- View All Users: Admins can see a list of all registered users.
- Search & Sort Users: Admins can search users by name and sort them by ID, name, username, or balance.
- History Page: Admins can check each user’s transaction history.

Tech Stack:
- HTML, CSS, JavaScript
- Java, Spring Boot, Spring Web, Spring Data JPA, Lombok
- MySQL
- REST APIs, Maven, IntelliJ IDEA
